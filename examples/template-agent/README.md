# Masumi Identity SDK — Template Agent

Runnable reference implementation showing how to use `@masumi_network/identity-sdk` end-to-end.

## Run

From the **monorepo root**:

```bash
pnpm install
pnpm --filter @masumi_network/identity-sdk build
pnpm --filter @masumi_network/identity-sdk-template-agent start
```

## What this demonstrates

_(To be expanded in Phase 4.)_

1. Linking an agent wallet to a KERI AID via the credential server
2. Requesting a Verifiable Credential from an issuer
3. Verifying a credential received during an agent-to-agent interaction
