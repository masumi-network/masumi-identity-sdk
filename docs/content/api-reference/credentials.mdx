Methods for issuing credentials to AIDs and working with credentials held by an AID. The three methods marked **pure** do not hit the network and are safe to call on the hot path.

## `issueCredential`

Issue an ACDC credential to an AID. The AID must already be connected via [`connectToAid`](/api-reference/agent-linking#connecttoaid).

```ts
issueCredential(params: IssueCredentialParams): Promise<{
  success: boolean;
  data: string;
}>
```

### Parameters

<ParamField path="params.schemaSaid" type="string" required>
  SAID of the schema the credential should conform to. Every credential type in
  Masumi is identified by its schema SAID.
</ParamField>

<ParamField path="params.aid" type="string" required>
  AID that will receive the credential. Must have been connected via
  `connectToAid` first.
</ParamField>

<ParamField path="params.attributes" type="Record<string, unknown>">
  Custom attributes to embed in the credential's `sad.a` block. Shape is
  schema-dependent â€” only fields defined in the schema are honored.
</ParamField>

### Returns

- `success` â€” `true` when the credential was issued.
- `data` â€” credential SAID or a server status message.

### Example

```ts
const result = await identity.issueCredential({
  schemaSaid: "EJ1gXWfzLyW2u0-YY5Z...", // Agent Verification schema
  aid: "EHpH79tPZoSl7VJZ7xMi3JWF4rH9wZ2ntGvKABd9N14z",
  attributes: {
    agentId: "agent-42",
    name: "Trip Planner",
  },
});

if (!result.success) {
  throw new Error(`Credential issuance failed: ${result.data}`);
}
```

## `getCredentialsForAid`

Fetch all credentials the credential server has issued to an AID.

```ts
getCredentialsForAid(aid: string): Promise<Credential[]>
```

### Example

```ts
const creds = await identity.getCredentialsForAid(
  "EHpH79tPZoSl7VJZ7xMi3JWF4rH9wZ2ntGvKABd9N14z",
);

console.log(`Agent holds ${creds.length} credential(s)`);
```

## `validateCredential` {#validatecredential}

<em>đź”’ pure â€” no network call</em>

Validate a credential's issuance status, revocation status, and expiration window.

```ts
validateCredential(
  credential: Credential,
  options?: CredentialValidationOptions,
): CredentialValidationResult
```

### Parameters

<ParamField path="credential" type="Credential" required>
  The credential to validate. See the [Credential type](/api-reference/types#credential).
</ParamField>

<ParamField path="options.expirationDays" type="number" default="365">
  How many days after issuance a credential is considered expired. Defaults to
  365.
</ParamField>

### Returns

A [`CredentialValidationResult`](/api-reference/types#credentialvalidationresult) with `isValid`, `status` (`"issued"` / `"revoked"` / `"expired"` / `"unknown"`), `message`, and optional `details`.

### Example

```ts
const result = identity.validateCredential(cred, { expirationDays: 90 });

if (!result.isValid) {
  throw new Error(`Credential not trusted: ${result.status} â€” ${result.message}`);
}
```

## `formatCredential` {#formatcredential}

<em>đź”’ pure â€” no network call</em>

Project a raw [`Credential`](/api-reference/types#credential) into a structured, display-friendly shape.

```ts
formatCredential(credential: Credential): FormattedCredential
```

### Example

```ts
const formatted = identity.formatCredential(cred);

console.log(formatted.credentialType);   // "AgentVerification"
console.log(formatted.issueeAid);        // "E..."
console.log(formatted.issuanceDateTime); // "2026-04-20T..."
console.log(formatted.status);           // "issued" | "revoked" | "expired"
console.log(formatted.attributes);       // { agentId: "agent-42", name: "..." }
```

## `findCredentialBySchema` {#findcredentialbyschema}

<em>đź”’ pure â€” no network call</em>

Find the first credential in an array matching a given schema SAID.

```ts
findCredentialBySchema(
  credentials: readonly Credential[],
  schemaSaid: string,
): Credential | undefined
```

### Example

```ts
const allCreds = await identity.getCredentialsForAid(aid);

const agentVerification = identity.findCredentialBySchema(
  allCreds,
  "EJ1gXWfzLyW2u0-YY5Z...", // Agent Verification schema SAID
);

if (!agentVerification) {
  throw new Error("Agent is not Masumi-verified");
}
```
