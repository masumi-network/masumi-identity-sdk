## `MasumiIdentityConfig`

Config passed to the [`MasumiIdentity`](/api-reference/masumi-identity) constructor.

```ts
interface MasumiIdentityConfig {
  credentialServerUrl: string;
  keriaUrl: string;
  fetch?: typeof fetch;
}
```

<ParamField body="credentialServerUrl" type="string" required>
  Base URL of the Masumi KERI credential server.
</ParamField>
<ParamField body="keriaUrl" type="string" required>
  Base URL of the KERIA agent. Required for signature verification.
</ParamField>
<ParamField body="fetch" type="typeof fetch">
  Optional `fetch` override. Defaults to global `fetch`.
</ParamField>

## `Credential`

Raw ACDC credential as returned by the credential server.

```ts
interface Credential {
  sad: {
    d?: string;       // Credential SAID (unique ID)
    s: string;        // Schema SAID
    i: string;        // Issuer AID
    a: {
      i: string;      // Issuee AID
      dt: string;     // ISO 8601 issuance datetime
      [key: string]: unknown;
    };
    ri: string;       // Status registry SAID
  };
  schema?: {
    $id: string;
    credentialType?: string;
    title?: string;
    version?: string;
  };
  status: {
    s: "0" | "1";     // "0" = issued, "1" = revoked
    et?: string;
    dt?: string;
  };
  rev?: {
    s: string;
    dt: string;
  };
}
```

<Callout type="info">
`sad` stands for *Self-Addressing Data* — the core credential payload. Its SAID
(`sad.d`) is a cryptographic hash of its own contents.
</Callout>

## `FormattedCredential`

Structured projection of a `Credential`, returned by [`formatCredential`](/api-reference/utilities#formatcredential).

```ts
interface FormattedCredential {
  schemaSaid: string;
  credentialType: string;
  credentialTitle: string;
  issueeId: string;
  issueeAid: string;
  issuanceDateTime: string;
  credentialStatusRegistry: string;
  attributes: Record<string, unknown>;
  status: "issued" | "revoked" | "expired";
  isValid: boolean;
}
```

## `CredentialValidationResult`

Returned by [`validateCredential`](/api-reference/utilities#validatecredential).

```ts
interface CredentialValidationResult {
  isValid: boolean;
  status: "issued" | "revoked" | "expired" | "unknown";
  message: string;
  details?: {
    issuedAt?: string;
    revokedAt?: string;
    expiresAt?: string;
    schemaSaid?: string;
    credentialType?: string;
  };
}
```
